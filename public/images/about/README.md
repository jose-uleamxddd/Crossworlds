# About Us - Images

Coloca aquí las imágenes de la página "About Us" (Sobre Nosotros).

## Imágenes necesarias:

Las imágenes se mostrarán en un carrusel automático en la sección "Our Story".

### Requeridas:
- `01.jpeg` - Primera imagen del carrusel
- `02.jpeg` - Segunda imagen del carrusel
- `03.jpeg` - Tercera imagen del carrusel
- `04.jpeg` - Cuarta imagen del carrusel

Puedes agregar más imágenes numerándolas secuencialmente (`05.jpeg`, `06.jpeg`, etc.) y actualizando el código en `app/about/page.tsx`.

## Instrucciones:

1. Descarga las imágenes de "About Us" de Supabase
2. Renómbralas como `01.jpeg`, `02.jpeg`, `03.jpeg`, `04.jpeg`
3. Colócalas en esta carpeta (`public/images/about/`)

## Uso:

El carrusel está configurado para cambiar automáticamente cada 4 segundos. Las imágenes deben ser de buena calidad y preferiblemente en formato horizontal (landscape) para mejor visualización.
