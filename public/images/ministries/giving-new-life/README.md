# Giving New Life - Images

Coloca aquí las imágenes del ministerio "Giving New Life" descargadas de Supabase.

**Carpeta en Supabase:** `Imagenes/ministries/giving new life`

**Instrucciones:**
1. Descarga todas las imágenes de esa carpeta en Supabase
2. Renómbralas como: `01.jpeg`, `02.jpeg`, `03.jpeg`, etc.
3. Colócalas en esta carpeta

**Nota:** La imagen `01.jpeg` será la imagen principal que se muestra primero.
