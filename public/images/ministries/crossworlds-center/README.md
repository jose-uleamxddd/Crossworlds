# Crossworlds Center for Connections - Images

Coloca aquí las imágenes del ministerio "Crossworlds Center for Connections" descargadas de Supabase.

**Carpeta en Supabase:** `Imagenes/ministries/Crossworlds Center for connections`

**Instrucciones:**
1. Descarga todas las imágenes de esa carpeta en Supabase
2. Renómbralas como: `01.jpeg`, `02.jpeg`, `03.jpeg`, etc.
3. Colócalas en esta carpeta

**Nota:** La imagen `01.jpeg` será la imagen principal que se muestra primero.
